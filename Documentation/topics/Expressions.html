LSystem Expressions

Overview

This is an overview of the expression syntax for the LSystem Engine. Expressions are used in "define" statements, parametric letters, and for conditions in productions.

in a "define" statement

define : theta : pi * 0.25
define : dist : 3 * cos(theta)

in a parameteric letter

+(theta + inc)
F(dist * n)

in a "production" condition

production :*: F(x) :*: x < 3 : F+F+
production : F(a) : F(x) : F(b) : x > 3 & a < b : F+F+

Variables

Variables are declared in "define" statements. Variable names may contain any combination of alpha-numeric characters and underscores, but variable names must start with a letter or an underscore.
Legal variable names:

_a = 3
_1a = 3
a_1 = 3
a1 = 3

(also legal...)

__a1x_y2_ = 3

Illegal variable names (start with any number):

1_a = 3
1a = 3

Operators

Additive Operators

+ (a+3) add
- (a-3) subtract

Multiplicative Operators

* (a*3) multiply
/ (a/3) divide
% (a%3) modulus

Assignment Operators

Standard assignment operators.
Please note: While these operators are provided, they are not recommended for use within any LSystem expression (Parametric Letters, Productions, Groups, Aliases, etc). The usual methodology to set values in a parametric system is in the passing of parameters. These operators can be used, but please understand the side-effects of doing so.

= (a=3) assignment
+= (a+=3) add assignment
-= (a-=3) subtract assignment
*= (a*=3) multiply assignment
/= (a/=3) divide assignment
%= (a%=3) modulo assignment

Logical Operators

All logic operations return either a one (1=true) or zero (0=false).

== (a == b) equals
!= (a != b) not equal to
> (a > b) greater than
≥ (a ≥ b) greater than or equal to
< (a < b) less than
≤ (a ≤ b) less than or equal to

Conditional Operators

| (a==b | a==c) or
& (a==b & a==c) and

Unary Operators

! (!a) not
- (-a) negative
++ (++a) increment (caution! this will set variable values)
-- (â€”a) decrement (caution! this will set variable values)

Nested expressions

Parentheses can be used to control evaluation. Expressions can be nested in sets of parentheses to any level.

(< expression> ) a = 3 * ( 4 - ( ( b * n ) / c ) ) + 1.75 )

Functions

General

sqrt(a) square root of a
pow(a,b) power of a to b
mod(a,b) modulus of a divided by b
exp(a,b) exponential of a to b

Log

log(n) natural (base e) logarithm of n
log10(n) base 10 logarithm of n

Trig Functions

cos(t) cosine of angle t
sin(t) sine of angle t
tan(t) tangent of angle t
cosh(t) hyperbolic cosine
sinh(t) hyperbolic sine
tanh(t) hyperbolic tangent
acos(t) angle where cosine is t
asin(t) angle where sine is t
atan(t) angle where tangent is t
atan(y,x) angle where tangent is the quotient y x

Angular Units

deg(r) Converts radians to degrees
rad(d) Converts degrees to radians

Limits & Rounding

ceil(x) smallest whole number from x
floor(x) largest whole number from x
round(x) round x to nearest whole number
round(x,d) round x to decimal places d

Random

rnd() Random float from 0 to 1
rnd(x,y) Random float in range of x to y
rndint(x,y) Random integer in range of x to y
